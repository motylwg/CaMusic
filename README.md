## Complexity

This project is mostly just a playground for learning Scala, and is currently in it's very early stages.

The usual suspects, such as [Project Euler](https://projecteuler.net/)
and [Ninety-Nine Scala Problems](http://aperiodic.net/phil/scala/s-99/),
worthy though they are, have not proved to be enough of an itch to scratch.

But there is something fascinating about complexity...
